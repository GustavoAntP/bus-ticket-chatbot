# Sistema Inteligente de Autoatendimento e Reserva de Passagens Rodoviárias

Chatbot inteligente para consulta, reserva e venda de passagens rodoviárias, com foco em automação conversacional e uso de inteligência artificial para interpretação de mensagens em linguagem natural.

## Sobre o projeto

Este projeto tem como objetivo desenvolver um sistema conversacional capaz de auxiliar usuários na busca de viagens rodoviárias, visualização de horários, seleção de assentos e realização de reservas por meio de um chatbot.

A proposta também prevê um painel administrativo para gerenciamento de viagens, reservas e acompanhamento de vendas, permitindo apoiar empresas ou parceiros que desejam oferecer atendimento automatizado.

## Objetivo geral

Desenvolver um sistema de reservas de passagens rodoviárias baseado em chatbot com inteligência artificial, capaz de interpretar mensagens dos usuários, apresentar opções de viagem e permitir a realização de reservas de forma automatizada.

## Funcionalidades previstas

- Consulta de viagens por origem, destino e data
- Exibição de horários disponíveis
- Seleção de assentos
- Reserva de passagem
- Simulação ou integração de pagamento
- Respostas contextualizadas com IA
- Painel administrativo para gerenciamento de viagens e reservas
- Dashboard com indicadores de vendas

## Tecnologias previstas

As tecnologias ainda podem ser ajustadas ao longo do projeto, mas inicialmente estão previstas:

- **Frontend/Admin:** React
- **Backend:** Node.js, Express
- **Banco de dados:** PostgreSQL
- **IA/NLP:** OpenAI API
- **Integração de mensagens:** WhatsApp API
- **Controle de tarefas:** Trello
